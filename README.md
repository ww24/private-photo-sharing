Private Photo Sharing
=====================

仲間内での写真共有サービス

Twitter 認証で、特定の人にだけ写真を共有することが出来ます。

Requirements
------------
* Node.js 0.10
* MongoDB 2.6

Setup
-----
1. `npm install`
1. start mongod process

Start
-----
```
npm start
```

or

```
pm2 start processes.json
```
